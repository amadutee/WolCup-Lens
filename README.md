# World Cup Lens

World Cup Lens is a mock-first Next.js App Router prototype for World Cup match intelligence. It presents upcoming, live, and recent fixtures, match detail pages, transparent player ratings, and rankings for players, teams, and positions.

## Tech stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Mock JSON-style data in `src/data/mockData.ts`
- Provider interface in `src/lib/footballApi.ts` for a future football API integration

## Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run typecheck
npm run build
```
